Este proyecto tiene como objetivo desarrollar un análisis exploratorio y visual del **Yelp Open Dataset**, con el fin de identificar patrones y factores que influyen en el desempeño de los negocios.

A través del uso de herramientas de análisis de datos y visualización, se busca generar insights que apoyen la toma de decisiones basada en datos.
